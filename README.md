## Install
### requirements:
* python3
* pip3

### setup:
run the following commands inside of your terminal
```
git clone https://github.com/ken-soares/EddyT.git
cd EddyT
pip install -r requirements.txt
chmod +x main.py
```
to run, execute `main.py`
